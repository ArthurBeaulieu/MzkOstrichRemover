MzkOstrichRemover reporting tool

This page will properly display a JSON report made with the MzkOstrichRemover script.

There is currently two ways to generate a report from the script. You must use -sd or --scan --dump to generate an error report for a given library, or use -ad or --analyze --dump to review the meta analyzis of several scan dumps.

For more information on those commands, consult the README.md file, or the repository's wiki (ReadMe – Wiki).
